# Vue Vine Module

[English](./README.md)

在 Nuxt 中使用 Vue Vine。

## 快速开始

通过一个命令将模块安装到你的 Nuxt 应用中：

```bash
npx nuxi module add @vue-vine/nuxt
```

就是这样！现在你可以在 Nuxt 应用中使用 Vue Vine Module ✨

## 贡献

<details>
  <summary>本地开发</summary>

  ```bash
  # 安装依赖
  pnpm install

  # 生成类型存根
  pnpm run dev:prepare

  # 使用 playground 开发
  pnpm run dev

  # 构建 playground
  pnpm run dev:build

  # 运行 ESLint
  pnpm run lint

  # 运行 Vitest
  pnpm run test
  pnpm run test:watch

  # 发布新版本
  pnpm run release
  ```

</details>
